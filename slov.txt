Словесно-формульный алгоритм

Функция bool fileExists(const string& filename)
1. Начало;
2. Проверка наличия файла при помощи встроенной функции ifstream file(filename);
3. Возврат результата при помощи file.good();
4. Конец.

Основная программа
1. Начало;
2. Инициализация генератора случайных чисел при помощи встроенной функции srand(time(0));
3. Объявление массива files с именами: "file1.txt", "file2.txt", "file3.txt";
4. Цикл по i от 0 до 2:
	5. Если файл files[i] не существует, тогда:
		6. Попытка создать и открыть файл для записи;
		7. Если файл успешно открыт, тогда:
			8. Цикл по j от 0 до 9:
				9. Генерация случайного числа от 1 до 10;
				10. Запись числа в файл с пробелом;
			11. Закрытие файла;
			12. Вывод "Файл <files[i]> успешно создан!";
		13. Иначе(из шага 7): вывод "Ошибка при создании файла <files[i]>";
	14. Иначе(из шага 5): вывод "Ошибка создания - файл <files[i]> уже существует";
15. Конец.
